# console — Консоль Оператора (Ф4)

Настоящая консоль оператора Merlin Forge Citadel. **Дисплей поверх готового бэкенда Ф3**
(CRM / договоры / биллинг / периоды) — денежную логику не дублирует, деньги считает ядро (#32).
Никакой бизнес-логики в UI.

## Стек
- **Vite + React + TypeScript**, **Tailwind v4** (CSS-first), UI-примитивы в паттерне **shadcn/ui**
  (hand-rolled: `src/components/ui/`, чтобы не зависеть от неподнятого shadcn MCP).
- Тема — **слаш** (midnight vault + gilded lines): токены из `_curator/design/slash-tokens`,
  опубликованы как утилиты Tailwind в `src/index.css` (`@theme`). Шрифты: Playfair Display + Inter.

## Запуск (локально)
```
npm install
npm run dev        # http://localhost:5173 (Vite, порт фиксирован)
npm run build      # tsc + vite build
npm run typecheck  # tsc (без эмита)
```
Dev-прокси `/api` → ядро (`MF_CORE_URL`, по умолчанию облачное). Токен оператора — `localStorage`
(`mfc.operator.token`); экран логина — отдельной подзадачей Ф4.

## Структура
- `src/components/layout/` — оболочка: `Sidebar` (навигация группами), `Header` (крошка/поиск/СТОП ФЛОТ), `AppShell`.
- `src/components/ui/` — примитивы: `button`, `card`, `badge`, `states` (пусто/грузится/ошибка).
- `src/screens/` — `Overview` (Обзор, по макету), `Placeholder` (разделы в сборке).
- `src/lib/` — `api` (клиент ядра), `nav` (карта навигации), `fixtures` (демо-данные Обзора), `useAsync`, `cn`.
