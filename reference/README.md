# reference/ — «Малыш Мерлин»

Замороженные страховки оригинала движка Пифагор на снимке **v8.3 @ `b75bd17`**
(полный хеш `b75bd176193abba1803899c092052c1f9c00a9eb`). Обоснование — ADR-0014.
**Ничего здесь не редактируется.** Правило платформы: с эталона КЛОНИРУЮТ, его не правят.

## 1. Профиль-эталон (в дереве)

`malysh-merlin-profile-v8.3.json` — заводские дефолты Пифагора @b75bd17: 23 рантайм-крутилки
(риск/капитал/исполнение/стратегия/ops) с типами и границами. Извлечён детерминированно из
`config.knobs` чистого клона (env чистый → код-дефолты), движок не запускался. Это baseline настроек
до появления Кузницы-UI (Ф5); полный залок-в-UI — Ф5, сейчас — замороженный read-only артефакт.

## 2. Полный архив кода (НЕ в дереве — release-ассет)

Полная копия репозитория Пифагора @b75bd17 (весь код, включая `engine/`, который в рантайм-картридж
не вендорили). Дизастер-рекавери: переживает даже исчезновение репо Пифагора.

| Поле | Значение |
|---|---|
| Файл | `malysh-merlin-v8.3-b75bd17.tar.gz` |
| Коммит | `b75bd176193abba1803899c092052c1f9c00a9eb` |
| Размер | ~1.5 MB (493 файла) |
| **SHA256** | `04f81c6196e76768ebf1eb6746a37d49a3f2668edc52c09a73d752ca8b5d4b34` |
| Как собран | `git archive b75bd17 --format=tar.gz --prefix=malysh-merlin-v8.3-b75bd17/` |

**Почему не в git-дереве:** архив 1.5 MB раздул бы историю навсегда, а git-lfs в проекте не
подключён (решение F1, DIRECTIVES #21). Хранится release-ассетом (ниже).

## 3. Где лежат независимые копии (страховка на 3 средах)

1. **Тег-указатель `malysh-merlin/v8.3-b75bd17`** в репо Пифагора `pifagor-v81` (annotated, вечный
   указатель на точный коммит; код Пифагора НЕ меняется) + GitHub release там же.
2. **Release-ассет Цитадели** — `malysh-merlin-v8.3-b75bd17.tar.gz` в GitHub Releases репозитория
   `merlin-forge-citadel` (Цитадель самодостаточна, не зависит от репо Пифагора).
3. **Локальный клон Оператора** `~/Desktop/pifagor-v81` — независимая копия вне GitHub (DIRECTIVES #21).

## 4. Инвариант «клонируй — не редактируй»

«Малыш Мерлин» (эталон · нулевой пациент · Пифагор v8.3) в платформе read-only. Любой эксперимент —
форк от него, всегда сравним с оригиналом. Enforcement в UI — Ф5 (Кузница). Версионирование: b75bd17 —
