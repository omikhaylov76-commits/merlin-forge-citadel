## 3. Recent Decisions Made (Session 1)

### D1: Платформа отделена от движков Контрактом Бота
**Context:** изначальный запрос «сервис поверх pifagor-v81» Оператор развернул: бот — личный, сервис — с нуля.
**Decision:** платформа не знает, что внутри бота; движки — Docker-картриджи за Контрактом (вход: конфиг+секреты; выход: телеметрия; команды start/pause/stop_close). Пифагор — картридж №1 через обёртку, его репозиторий не трогаем.
**Rationale:** 1) несколько движков в будущем; 2) развитие движка не ломает платформу; 3) модульность — принцип Оператора.
**Documented in:** wiki/decisions/0001, 0002.

### D2: Railway v1 через InfraDriver
**Decision:** хостинг v1 — Railway (GraphQL API: serviceCreate из образа + variables). Драйвер оркестрации абстрагирован; план Б — DockerDriver/VPS.
**Rationale:** Оператор уже живёт на Railway; API покрывает оркестрацию; выход дешёвый. Факты и цены проверены 2026-07 (см. wiki/entities/railway.md). Static IP общий → риск принят, пересмотр на mainnet-объёмах. **Pro-план НЕ включать до Ф1–2.**
**Documented in:** wiki/decisions/0003.

### D3: Секреты — конверт-шифрование (0004); D4: клиентский стоп двухступенчатый — Пауза мгновенно, Стоп-и-закрыть с двойным подтверждением и алертом (0005); D5: труба под бота-дирижёра — ансамбли + токены со скоупом группы, UI в v1 нет (0006).

### D6: Имя — Merlin Forge Citadel
«Fleet» отклонён Оператором в пользу «Citadel». Отличается от его смежного проекта merlins-forge.

### D7: База знаний — вики по паттерну Карпати
