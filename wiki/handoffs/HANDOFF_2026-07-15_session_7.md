## 3. Решения сессии
- **AUM = телеметрия-снимок последней equity ТОЛЬКО по активным** (running+paused; stopped/terminated исключены —
  деньги возвращены клиенту). Дисплей-метрика обзора ≠ авторитетное биллинг-equity. Ратифицировано #38/#42.
- **«Комиссия начислена»** без invoiced-флага для v1 (#38б): начислено==к выставлению до запуска; точный
  `invoiced_at` — отдельной биллинг-задачей позже (деньги → независимое ревью).
- **Конфиг-diff без дрейфа:** safe-mode env (BYBIT_DEMO/LIVE_TRADING/ключи) 23 крутилки НЕ трогает → воркер грузит
  код-дефолты = эталон. Демо-значения старого constructor-knobs были из тюнингованного Railway-env, не из deploy.
- **Движок Пифагора локально НЕ гоняли** (тянет Bybit-demo + рыночные данные = Шаг B/облако). БД воркера
  инициализировали ЕГО ЖЕ кодом (`DB owner=True` схема + `Ledger.seed(10000,10000)`=equity 20000 + пульс),
  дальше РЕАЛЬНЫЙ адаптер картриджа гонял S4-контракт. Контракт+конфиг доказаны faithfully.
- ghcr держим **приватным** (Шаг B, registry-cred). Реальная торговля = go-live гейт.
