# ADR-0004. Секреты: шифрование конвертом
Решение: ключи бирж хранятся в БД шифрованными (per-record data key, обёрнутый master-key).
Master-key — только в env оркестратора. Расшифровка — в памяти оркестратора в момент старта
инстанса; передача боту — env при создании сервиса. Ротация: смена master-key = перешифровка
data-keys, ключи бирж не пересоздаются. В аудит пишется факт доступа к ключу (кто/когда/зачем).
Отклонено: Vault (котлован для этого масштаба), plaintext в env ядра (недопустимо).
