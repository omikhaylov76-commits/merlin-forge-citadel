# ADR-0006. Труба под дирижёра (ансамбли)
Решение: в домене — ensemble_id у инстансов + таблица ensembles; в API — токены со скоупом
«читать телеметрию группы / слать команды группе». Дирижёр = обычный тип бота с таким токеном.
В v1 UI ансамблей нет (заглушка), механизм в фундаменте есть. Прямых связей бот↔бот не бывает.
