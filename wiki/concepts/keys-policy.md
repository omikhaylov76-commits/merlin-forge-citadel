# Политика ключей клиента
Выпуск: клиент создаёт API-ключ сам по нашей инструкции: права ТОЛЬКО торговля (spot/derivatives
по потребности движка), вывод — запрещён, IP-whitelist — адреса платформы. Передача — через
одноразовую ссылку-форму (не мессенджер!). Хранение — ADR-0004. Проверка прав ключа —
автоматически при подключении и раз в сутки (perms_checked_at). Ротация/отзыв — runbook в 40_operations.
