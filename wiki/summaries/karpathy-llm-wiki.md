Суть: знание проекта — не переписка, а живая вики, которую LLM ведёт как дисциплинированный
библиотекарь. Слои: immutable источники (sources/) → генерируемые страницы (concepts/entities/
summaries) → index.md (каталог с однострочниками) + log.md (append-only журнал операций).
Правила ведения — в CLAUDE.md (schema-слой). Рабочие циклы: Ingest (новый источник → конспект +
обновить связанные страницы + index + log), Query (ответы из вики с ссылками; ценное — обратно
страницей), Lint (регулярная сверка: противоречия, сироты, битые ссылки, устаревшее).
Философия: «вики — накапливающийся артефакт»; человек курирует, LLM ведёт бухгалтерию знаний.
Принято за стандарт БЗ Цитадели (см. CLAUDE.md «Вики»).
