Торговый движок Оператора (отдельный репозиторий pifagor-v81, Цитадель его НЕ хранит и не меняет).
Суть: only-long сетки от 4h-пробоя с Фибо-ногами, стоп вшит в ордер, kill-switch −50%, parity-закон
(живое исполнение = движку бэктеста), demo-first, ~900 тестов. Станет картриджем №1 через
обёртку-образ с адаптером Контракта (ADR-0001, ADR-0002). Интерфейс брокера внутри — ~15 методов,
семантика Bybit-специфична — это главная работа при мультибирже.

## Аксиома пере-якоря (V3 — окончательная; подтв. Оператором 2026-07-23)
Сетап = 3 buy-лимитки на откате импульса **A (низ) → B (верх)**: 0.382 (ближняя к B) · 0.5 (середина) ·
0.618 (дальняя). **Якорь ЖИВОЙ, пока сетап не закоммичен:**
1. **Новый хай → пере-якорь ВСЕГДА.** Пока не закоммичен, любое обновление хая выше B двигает B на новый
   хай, все 3 ордера перестраиваются по новым фибо-ценам (0.382/0.5/0.618 от A→новый B). Якорь едет за ценой.
2. **После скальпа первой ноги (0.382) пере-якорь ПРОДОЛЖАЕТ двигаться** на новый хай (снятая нога
   пере-взводится). Скальп НЕ замораживает якорь.
3. **Заморозка — ТОЛЬКО при касании второй ноги (0.5) = COMMIT.** Дошла до 0.5 → «импульс отработал»:
   пере-якоря больше нет, ордера ведутся как есть до конца; новый сетап с новыми A/B — только ПОТОМ.

Триггер заморозки — **именно касание 0.5**, НЕ факт залива ноги и НЕ скальп.
**Реализация:** `execution/lifecycle.py` — commit=залив 0.5 (`COMMIT_LV`, :191); блок пере-якоря
`if not committed` (:256); all-pending+хай (:265); после-скальпа+хай при `REANCHOR_AFTER_SCALP=True` (:269).
**V3 = `REANCHOR_AFTER_SCALP=True` = эта аксиома. V2 (False) = устаревшая, ошибочно морозит на скальпе.**
Флаг — env (`config/knobs.py`); **⚠️ вступает в силу в ЖИВОМ процессе только при РЕДЕПЛОЕ** (переменная ≠
работающий процесс). Борс переведён на V3 (2026-07-23). Малыш Мерлин (эталон @b75bd17, ADR-0014) — V2 by
design, замороженный: менять = амендмент ADR-0014, не по чату. Раскатка V3 по флоту — решение Куратора.
Пере-якорь движок в лог/журнал НЕ пишет → аудит только по смене сетки ордеров (довод за Сигнальный журнал, вар. C).
