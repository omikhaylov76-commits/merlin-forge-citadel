# Merlin Forge Citadel — Кузница Мерлина, флотская платформа

Платформа управления флотом торговых ботов: админ-консоль (CRM + мониторинг + исследования),
оркестрация клиентских инстансов, read-only портал клиента. Движки ботов — сменные картриджи
за Контрактом Бота; первый картридж — движок Пифагор V8.1 (живёт в СВОЁМ репозитории, здесь не хранится).

## Три закона платформы
1. **Платформа не знает, что внутри бота.** Общение только через Контракт Бота (`contracts/`).
2. **Секреты не покидают оркестратор.** Ключи бирж шифруются конвертом, в логи/API/портал не попадают никогда.
3. **Модули общаются только через API и контракты.** Никаких прямых связей, которые потом рвать по живому.

| `infra/` | Compose, миграции, env-шаблоны, CI | скелет |
